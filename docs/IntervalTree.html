    IntervalTree

    Implementation of interval binary search tree
    Interval tree stores items which are couples of {key:interval, value: value}
    Interval is an object with high and low properties or simply array of numeric [low,high] values
    If interval is an object, it should implement and expose methods less_than, equals_to, intersect and others, see documentation
